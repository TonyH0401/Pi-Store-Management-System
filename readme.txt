Install:
1. Locate folder "Database" -> Find file "NET_MIDTERM.sql"
2. Open MSSQL -> Open the "NET_MIDTERM.sql" -> Run the file
3. Locate folder "Midterm-NET" -> Locate App.config -> Change DataSource to your local machine's database on MSSQL -> Catalog = "NET_MIDTERM"
4. Run WinForm -> Locate folder "Midterm-NET" -> Locate *.sln file -> Open -> Start
or
4. Locate bin -> Locate Debug -> Run Application "Midterm-NET"
or
4. Locate bin -> Locate Release -> Run Application "Midterm-NET"
